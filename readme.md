# Tkinter Login System

This is a simple login system GUI built using Tkinter in Python. It allows users to log in using their username and password.

## Features

- User Login
- Simple and Intuitive Interface
- Basic Error Handling for Incorrect Login Credentials

## Requirements

To run the application, ensure you have Python installed on your system.

## Getting Started

1. **Clone the Repository**

   ```sh
   git clone https://github.com/SudeepAcharjee/Tkinter-auth
   cd Login-tkinter(GUI)
   ```

2. **Run the Application**

   ```sh
   python auth.py
   ```

## Usage

1. **Login**

   Upon running the application, you will be presented with a login screen. Enter your username and password and click the "Login" button.

2. **Error Handling**

   If you enter incorrect login credentials, an error message will be displayed.

3. **Exiting the Application**

   You can exit the application by clicking the close button on the window or using your system's window manager to close the application.
